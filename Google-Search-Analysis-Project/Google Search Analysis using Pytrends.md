# Google Search Analysis using Pytrends

> A data analysis project that explores Google search trends using Pytrends to uncover user behavior and keyword popularity.

---

## Project Overview

In this project, I analyzed **Google search trends** using the Pytrends library to understand how user interest in specific keywords changes over time and across different regions.

The project focuses on extracting and visualizing:

- keyword popularity  
- country-wise search trends  
- time-based interest patterns  
- related keyword comparisons  

This helps in understanding user behavior and market demand.

---

## Problem Statement

The objective of this project is to:

- analyze search trends for different keywords  
- identify top countries where a keyword is most searched  
- understand how interest changes over time  
- compare related keywords  
- visualize global search behavior using data  

---

## Data Source

The data is collected dynamically using:

- **Google Trends API (via Pytrends)**  

The following data is extracted:

- Interest over time  
- Interest by region (countries)  
- Related queries and keywords  

---

## Key Tasks Performed

### 1. Dynamic Keyword Search
- Built a system where keywords can be changed easily  
- Allows analysis of multiple search terms  

### 2. Top Countries Analysis
- Extracted top 15 countries where the keyword is most searched  
- Visualized using bar charts  

### 3. Global Map Visualization
- Created a world map showing keyword popularity across countries  

### 4. Time-Based Trend Analysis
- Analyzed how interest in a keyword changes over time  
- Identified peaks and seasonal trends  

### 5. Keyword Comparison
- Compared multiple related keywords  
- Visualized trends using line charts  

---

## Exploratory Data Analysis (EDA) Insights

- Certain keywords show **seasonal trends** (peak at specific times)  
- Some countries dominate search volume for specific keywords  
- Trends vary significantly across regions  
- Keyword popularity changes based on real-world events
